   # 1. Introduction
   The blinking happens because the circuit generates a time delay signal.
LED ON → wait → OFF → wait → repeat
Circuit gives voltage → LED turns ON

A timing element (like capacitor or program delay) starts charging
After some time → signal changes → LED turns OFF
Process repeats again
